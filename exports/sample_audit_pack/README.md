# Sample audit pack

This directory contains human-readable sample deliverables for the Phase 2 pilot release.

## Included files

- `proofops-investigation-pack.pdf` — founder-led ProofOps investigation pack
- `govai-proof-pack.pdf` — GovAI pilot proof pack
- `foodqa-proof-pack.pdf` — FoodQA pilot proof pack
- `index.md` — audit pack front matter and summary
- `evidence_register.csv` — concise evidence inventory
- `review_timeline.md` — decision trail and disposition summary

## Pack structure

Each sample proof pack now includes:

- cover page
- executive summary
- evidence register
- review timeline
- AI accountability section where applicable
- deviation or finding summary
- export manifest
- hash list
- reviewer decision record
