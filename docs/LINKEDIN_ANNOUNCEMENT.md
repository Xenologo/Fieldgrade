# LinkedIn announcement

Fieldgrade ProofOps is now available for private alpha pilots.

Fieldgrade is a local-first evidence-governance workspace for teams that need to capture files, seal provenance, review decisions, govern AI-assisted work, and export proof packs.

Current alpha focus:
- audit-supporting proof-pack workflows
- human-reviewed AI accountability records
- reproducible pilot installs
- founder-led ProofOps setup

This is pilot-ready software for controlled evaluations, not a full ERP, full QMS, or automated compliance product.
