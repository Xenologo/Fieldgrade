# AI-use disclaimer

Fieldgrade can record AI-assisted work, but it does not convert AI output into trusted fact by itself.

## Pilot-safe position

- AI output is a review aid
- human review is mandatory
- model and version context should be recorded where possible
- prompts and outputs may contain sensitive information and should be handled accordingly
- pilot users decide whether external AI endpoints are permitted

## Non-claim statement

Fieldgrade is audit-supporting and evidence-governance oriented. It is not an AI auditor, automated compliance engine, or regulator-approved assessment product.
