# Launch post

**Fieldgrade ProofOps is now available for private alpha pilots: a local-first evidence governance workspace for teams that need to capture files, seal provenance, review decisions, govern AI-assisted work, and export proof packs.**

Fieldgrade is built for teams that need an audit-supporting proof layer without jumping straight to a heavyweight ERP or full QMS programme. The alpha release focuses on practical pilot value: reproducible installs, sample proof packs, screenshots, workflow templates, and a founder-led setup path.

What the alpha is for:

- private evaluation
- founder-led setup
- pilot deployments
- human-reviewed proof-pack workflows

What the alpha is not:

- a self-serve commercial SaaS
- a regulator-approved product
- an automated compliance system
- a replacement for qualified review or audit roles
