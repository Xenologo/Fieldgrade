# FoodQA pilot workflow template

## Target buyer

- small food producer
- beverage producer
- quality consultant
- contract packer
- technical QA team

## Minimum workflow objects

1. **Batch or event record**
   - batch ID
   - product
   - date
   - operator
2. **Supplier evidence register**
   - COA
   - supplier declaration
   - specification file
3. **Deviation note**
   - issue observed
   - impact
   - containment action
4. **CAPA-lite note**
   - corrective step
   - responsible person
   - due date
5. **Calibration or equipment evidence**
   - asset
   - calibration date
   - supporting certificate
6. **Review decision**
   - reviewer
   - status
   - release decision
7. **Audit-pack export**
   - summary page
   - evidence register
   - finding summary
   - reviewer record

## Proof-pack focus

FoodQA pilots should make it easy to show how a batch decision was supported, reviewed, and packaged for audit-supporting use.
