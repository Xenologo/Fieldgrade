# Fieldgrade v0.9.0-alpha release notes

## Summary

Fieldgrade v0.9.0-alpha is the first release candidate aimed at private pilots and founder-led custom setup engagements. It packages the current evidence-governance substrate with clearer installation, trust, and operational guidance.

## Included in this alpha

- buyer-facing product pages for Core, ProofOps, GovAI, and FoodQA
- a public demo journey and sample outputs
- Docker Compose deployment paths for local and single-host production use
- CI coverage for hygiene, compose config validation, docker build, and Python tests
- root docs for licensing, security reporting, installation, data handling, and release execution

## Intended use

- private alpha evaluations
- founder-led pilot deployments
- internal stakeholder demos

## Not yet included

- automated release artifact publishing
- dependency lockfiles
- self-serve payment and fulfillment automation
- polished screenshot/GIF sales collateral

## Release operator notes

- use `INSTALL.md` for the simplest setup path
- use `SECURITY.md` and `DATA_HANDLING.md` when briefing pilot users
- record the Docker smoke-test outcome before publishing a public alpha build
