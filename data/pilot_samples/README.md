# Fieldgrade pilot sample staging

This directory is intentionally empty except for repository placeholders.

Use it only for partner-approved, non-sensitive pilot samples that are explicitly permitted for public repository inclusion.

## Before adding any file

- Follow `docs/proposal/FIELDGRADE_PILOT_DATA_REPLACEMENT_PROTOCOL.md`.
- Confirm data owner, licence or publication permission, sensitivity class, redaction status, export permission, and reviewer sign-off.
- Keep only the minimum public-safe sample needed to support a pilot narrative or reviewer walkthrough.

## Do not add

- unredacted personal data,
- confidential customer, supplier, or research files,
- secrets or credentials,
- records without publication permission, or
- any sample that could be mistaken for unrestricted production evidence.

If a partner sample is useful for a private pilot but not safe for public release, keep it outside this repository.
