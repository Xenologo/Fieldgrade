Demo flow

Use simple labels and visible proof at every step.

Messy file arrives

Example: Supplier_COA_Batch_118.pdf or an AI prompt/output note.

Register the evidence

Visible fields: object ID, file name, evidence type, source, uploader, timestamp, SHA-256 fingerprint, and review state.

Seal provenance

Visible outputs: bundle ID, canonical hash, manifest, and provenance event chain.

Review the record

Visible states: Raw, Witnessed, Pending Review, Approved, Rejected, Quarantined, Exported.

Export the proof pack

Outputs: PDF proof pack, ZIP sample bundle, JSON manifest, checksum list, and reviewer decision record.
