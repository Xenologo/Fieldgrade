# FoodQA demo

This example demonstrates the first flagship vertical pack: turning ordinary food and beverage QA evidence into a reviewable, exportable proof trail.

## Included inputs

- `batch_record.csv`
- `deviation_note.md`
- `supplier_coa_summary.json`
- `calibration_register.csv`

## Storyline

1. Capture the batch record and supplier evidence.
2. Attach the deviation note and calibration context.
3. Route the review through the appropriate policy mode.
4. Export the audit pack and evidence bundle references.

## Expected buyer takeaway

Fieldgrade FoodQA does not replace the plant floor. It packages the evidence around the plant floor so audits, investigations, and supplier challenges can be answered quickly.
