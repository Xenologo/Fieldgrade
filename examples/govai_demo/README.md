# GovAI demo

This example demonstrates how the same Fieldgrade substrate can package AI accountability evidence.

## Included inputs

- `system_register.json`
- `prompt_log.md`
- `output_sample.md`
- `review_decision.md`

## Storyline

1. Register the AI-assisted use case.
2. Attach prompt/output evidence and model metadata.
3. Record the reviewer decision.
4. Export the GovAI accountability pack.

## Expected buyer takeaway

Fieldgrade GovAI turns disconnected policy statements and prompt snippets into a governed evidence trail that can survive audit, procurement, or internal challenge.
